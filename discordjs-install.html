Installing Discord.js

This tutorial will teach you how to set up your Discord.js enviroment ready
for coding a discord bot.



Step 1:
First you want to install Node.js from any of these buttons.

WINDOWS:



MAC:


LINUX:
If you want a .zip file or some other mean of downloading, get it from here.

Step 2:
Next you have to open the application that it insalled, called Node.js command prompt

Step 3:
Next you have to locate your workspace with cd and then type this command:
npm install discord.js --save if you don't want voice support
If you do want voice support then type either

npm install discord.js node-opus --save

or

npm install discord.js opusscript --save

Note:
You should see lots of things go by on the terminal, as long as there isn't a big old red thing on the screen,
you should be alright!

And now your done!
How easy was that? Now you are ready to start coding your discord.js bot!
